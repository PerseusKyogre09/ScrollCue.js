ScrollMagic

A lightweight, customizable scroll animation library

Introduction

ScrollMagic is a powerful yet simple animation library inspired by "The Boat" interactive story. It allows you to create beautiful scroll-based animations with minimal code.

Simply include the JavaScript and CSS files in your HTML, add data attributes to your elements, and watch them come to life as you scroll.

Scroll down to see more animations!

Getting Started

To get started with ScrollMagic, follow these simple steps:

1. Include the CSS and JavaScript files:

<link rel="stylesheet" href="ScrollMagic.css"> <script src="ScrollMagic.js"></script>

2. Initialize the library:

<script> // Initialize with default options const scrollMagic = new ScrollMagic(); scrollMagic.init(); // Or with custom options const scrollMagic = new ScrollMagic({ offset: 0.1, // When to trigger (10% of viewport) threshold: 0.1, // Intersection threshold once: true, // Only animate once }); scrollMagic.init(); </script>

3. Add data-scroll attributes to your elements:

<div data-scroll="fade-up">This element will fade up</div> <div data-scroll="zoom-in" data-scroll-duration="1000">This element will zoom in</div>

Customization

ScrollMagic is highly customizable. You can adjust animation duration, delay, easing function, and more using data attributes:

<div data-scroll="fade-up" data-scroll-duration="2000" /* Duration in ms */ data-scroll-delay="500" /* Delay in ms */ data-scroll-easing="ease-in-out" /* CSS easing function */ data-scroll-offset="70" /* Custom offset percentage */ data-scroll-once="false" /* Override global once setting */ data-scroll-color="#ff0000" /* Custom color */ data-scroll-intensity="100" /* Animation intensity */ data-scroll-speed="2" /* Animation speed */ data-scroll-callback="myFunction" /* JavaScript callback */ > Customized Animation </div>

Advanced Features

Adding Custom Animations

You can add your own custom animations to the library:

scrollMagic.addAnimation('my-animation', { initial: 'transform: skewX(30deg); opacity: 0;', visible: 'transform: skewX(0); opacity: 1;', keyframes: ` 0% { transform: skewX(30deg); } 100% { transform: skewX(0); } ` });
